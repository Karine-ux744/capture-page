# Jornada Python da Hashtag

- Landing Page para Evento de Programação
- Página de inscrição para curso gratuito de Python

## Features

- [x] Formulário de inscrição com popup modal
- [x] Validação de campos (nome, email, WhatsApp)
- [x] Seção de passo a passo do processo de inscrição
- [x] Preview do conteúdo das 4 aulas
- [x] Seção sobre o instrutor
- [x] Design responsivo para mobile e desktop
- [x] Mensagens de sucesso/erro no envio do formulário

## Usage

```bash
$ git clone git@github.com:Karine-ux744/capture-page.git

$ cd capture-page
$ # Abra o arquivo index.html no navegador
$ # Ou use um servidor local como Live Server (VS Code)
```
